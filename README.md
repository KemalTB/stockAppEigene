PNPM KURULUMU
https://pnpm.io/installation

microsoft: 
npm install -g pnpm

mac: 
sudo npm install -g pnpm
(mac: bilgisayar sifresini isteyecek)

kurulum basarili olmusmuyu kontrol icin: 
pnpm -v


terminali temizlemek icin:
clear
